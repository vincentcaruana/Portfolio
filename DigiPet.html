Digital Pet Interface

UI/UX Design, Figma, p5.js, Illustration, 3D Modeling

This project is a high-fidelity interactive prototype of a divine remote control designed to manage and influence an artificial, defenseless colony of creatures known only as Frank. Using Figma's prototyping features, the interface explores the ethical challenges of absolute control, placing the user at the center of a moral decision-making system.
